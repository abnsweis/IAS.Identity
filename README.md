# 🔐 Identity & Access Management System

A secure backend system built using ASP.NET Core for managing users, authentication, authorization, roles, and permissions.

---

## 📌 Overview

This project is an Identity and Access Management system designed to handle user authentication and authorization in a secure and scalable way.  
It provides a complete role-based access control system using JWT authentication.

---

## 🚀 Features

- User registration and login system  
- JWT authentication and authorization  
- Role-based access control (RBAC)  
- Permission management system  
- Assign and remove roles from users  
- Activate and deactivate user accounts  
- Refresh token support  
- Secure authentication flow  

---

## 🧠 Technologies Used

- ASP.NET Core Web API  
- Entity Framework Core  
- SQL Server  
- JWT Authentication  
- Clean Architecture (Layered Structure)   

---

## 🔐 Security Features

- JWT-based authentication  
- HttpOnly refresh tokens  
- Role-based authorization  
- Secure password hashing  
- Token revocation support  

---

## 🏗 Architecture

The project follows a layered architecture:

- API Layer (Controllers)  
- Application Layer (Services)  
- Domain Layer (Entities & Core logic)  
- Infrastructure Layer (Database & external services)  

---

## 🎯 Purpose

The goal of this project is to centralize authentication and authorization logic, making it reusable, secure, and scalable for real-world applications.
